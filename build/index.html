Build Time

动手创造与项目制学习

孩子不是被动听课，而是要把想法变成作品。Build 模块让孩子设计未来城市、搭建任务模型、规划机器人行动、构思无人机探索路线，也可以把 3D 模型、机械臂、积木结构和未来工坊结合起来。

主题介绍

创造不是装饰环节，而是学习的核心证据。

孩子会在每个主题后完成一个可见作品：月球补给站、情绪机器人、无人机救援路线、未来教室模型、能源城市地图。作品不追求复杂昂贵，而追求想法清楚、结构合理、能被讲明白。

教学方式

从任务卡到原型，再到展示。

Build Time 采用项目制路径：理解任务、画出草图、选择材料或数字工具、制作原型、测试问题、修改方案、讲述结果。孩子会意识到失败不是扣分，而是工程设计里的重要信息。

Core Abilities

培养孩子把想法落地的能力。

创造力

从一个未来问题开始，提出自己的解决方案。

工程思维

关注结构、功能、限制和迭代。

动手能力

通过搭建、绘制、建模和测试把想法做出来。

项目管理能力

理解目标、步骤、时间和展示结果。

解决问题能力

面对作品不稳定、逻辑不清时寻找调整方案。

团队协作力

分工、讨论、倾听，并把不同想法整合成作品。

For Parents

家长能看懂的课程价值

Build 模块把孩子的学习从“我听懂了”推进到“我能做出来并解释”。这类能力会影响 STEM 学习、项目展示、领导力和长期自信。孩子在小作品里练习的是未来社会真实需要的创造流程。

Kids Will Say

学生会觉得很酷的表达

“这是我设计的月球补给站，它能保护植物。”“我的无人机路线会避开风暴区。”“我修改了机械臂，因为第一版抓不稳。”
AI 时代教育理念

AI 可以给建议，但孩子要成为设计者。

在 Build 模块里，AI 可以帮助孩子生成灵感、检查遗漏、模拟不同方案，但孩子需要选择目标、解释取舍、承担修改。真正重要的是从使用工具走向创造系统。
